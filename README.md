# Mohloling Academy of Arts Website

This is a static one-page website designed for Mohloling Academy of Arts.

## Open offline
Double-click `index.html` or open it with Chrome.

## Host on GitHub Pages
1. Create a new GitHub repository called `mohloling-academy-of-arts`.
2. Upload `index.html` to the repository root.
3. Go to Settings → Pages.
4. Set source to `Deploy from a branch`, branch `main`, folder `/root`.
5. Save and wait for GitHub Pages to publish.

## Edit before launch
Search inside `index.html` for:
- prices in Maloti
- instructor placeholders
- event dates
- payment details
- social links
